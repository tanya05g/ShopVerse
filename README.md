# 🛍️ ShopVerse

**Live Demo:** https://shop-verse-omega-seven.vercel.app/

**GitHub Repository:** https://github.com/tanya05g/ShopVerse

## 📌 Overview

ShopVerse is a modern full-stack e-commerce web application designed to provide a seamless online shopping experience. The platform features an intuitive user interface, responsive design, secure authentication pages, product browsing, shopping cart functionality, wishlist management, and a smooth checkout process.

Built using modern web technologies, ShopVerse demonstrates scalable architecture, server-side rendering capabilities, and efficient frontend development practices.

---

## ✨ Features

- Modern and responsive landing page
- Product listing and browsing
- Product details page
- Shopping cart management
- Wishlist functionality
- User login and registration
- Secure checkout process
- Order confirmation page
- About, Contact, FAQ, Privacy Policy, and Terms pages
- Mobile-friendly design
- Server-Side Rendering (SSR)
- Fast and optimized performance

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/tanya05g/ShopVerse.git
```

### Navigate to the project directory

```bash
cd ShopVerse
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### Build the project

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
